Purpose, goals and audience

The purpose of this website is to be both a connection for table tennis interested students, and the organization, as well as to be an information hub for the organization. Therefore, our primary goal is to provide users with the opportunity to join open training/tryouts, and join the organization, so that the organization can get new active members. therefore, we will provide contact information, and the training schedule in a neat manner.


Minimum Requirements (Should not be part of delivery)

The minimum requirements are to make at least four JS scripts. We plan on making seven:

1. scoreboard.js (Scoreboard)

A script to dynamically display information to a scoreboard. It will sort the list of players with their scores from highest to lowest score.

2. training_table.js (Training table)

A script to dynamically display events and training times. It will be displayed in a weekly calendar, where the user will be able to scroll through the weeks.

3. form_validation.js (Form validation)

A script for checking whether or not the filled in sign up form is valid.

4. hamburger_navbar.js (Hamburger navbar)

A script for a button that reveals links to the different pages on the site. This button will be in the top right in the navbar.

5. homepage_gallery.js (Homepage gallery)

In the homepage the user will be met with an image. This image should switch to the next image in the gallery after a certain amount of time.

6. gatherings_carousel.js (Carousel for social gatherings page)

In the social gatherings page with events and tournaments we will create a carousel, with an element for each year, where we have information about that year's social events. For this we will need js for the carousel itself, ensuring smooth scrolling, and elements that snap to the center of the page, as well as accordion elements for each evenr per year.

7. dark_mode.js (Dark mode/Light mode toggle)

There will be a button to toggle between dark and light mode.
